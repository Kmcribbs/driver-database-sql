# Driver Database Design Using SQL

## Overview
This project demonstrates the creation of a relational database table for managing driver information. The table includes key attributes such as driver ID, name, license status, and performance ratings.

## Skills Used
- SQL (CREATE TABLE)
- Database design
- Data structuring

## Key Features
- Designed a structured table for storing driver data
- Implemented a primary key to uniquely identify each driver
- Defined appropriate data types for efficient data storage

## File Included
- driver_table.sql: SQL script used to create the driver database table
